💬  Sobre o projeto 
O Alurakut é uma aplicação web clone do antigo Orkut, porém traz uma interface mais moderna com tecnologias distintas. 


💻   Como utilizar?

Para utilizar o Alurakut, basta [clicar aqui ](https://alurakut-blond-psi.vercel.app/)
